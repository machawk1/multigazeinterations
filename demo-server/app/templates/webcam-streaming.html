The following demo of the Heuristic based Multi-gaze interactions have been ported from the original desktop based implementation to support web cam driven multi user interactions through web browsers. The performance of the application can vary with the speed of internet connection (Processing is done only in the server-side) and the quality of the web camera. In the current implementation, we perform the gaze recognition in sequence( capture, process, show, capture next), as a result the video stream has no continuous nature.
